# Paster

macOS 剪貼簿歷史管理工具。自動記錄複製的內容，快速搜尋、釘選常用片段、分類管理。

## 功能

- **剪貼簿自動監聽** — 文字、圖片、URL、檔案路徑
- **全域熱鍵** — 預設 `⌘⇧V` 呼叫主視窗（可在設定中自訂）
- **卡片式瀏覽** — 格狀卡片 + 右側預覽面板
- **即時搜尋** — 輸入即篩選，支援依類型和來源 App 篩選
- **快速貼上** — 雙擊或 Enter，自動切回前一個 App 貼上
- **釘選 & 分類** — 常用片段可釘選並分類管理（一般、程式碼、網址等）
- **鍵盤操作** — `↑↓` 選擇、`⌘1~9` 快速貼上、`⌃Tab` 切換分頁
- **隱私保護** — 自動排除密碼管理器內容
- **自動清理** — 可設定保留天數和最大筆數

## 系統需求

- macOS 13 (Ventura) 以上
- Apple Silicon (arm64)

## 安裝

### 方法一：直接下載

到 [Releases](../../releases) 下載最新的 `Paster.app.zip`，解壓後拖到 `/Applications`。

### 方法二：從原始碼編譯

需要 macOS Command Line Tools（不需要完整 Xcode）：

```bash
git clone https://github.com/lukehsuhao/paster.git
cd paster
chmod +x build.sh
./build.sh
open ClipStash.app
```

> 注意：編譯產出的 .app 目錄名稱仍為 `ClipStash.app`，但視窗標題和功能名稱已改為 Paster。

## 使用

1. 啟動後會在選單列出現剪貼簿圖示
2. 按 `⌘⇧V`（或自訂熱鍵）開啟主視窗
3. 複製任何東西後會自動記錄
4. 雙擊或按 Enter 貼上到前一個 App

### 快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `⌘⇧V` | 開啟 Paster（可自訂） |
| `↑` `↓` | 選擇項目 |
| `Enter` | 貼上選取項目 |
| `⌘1` ~ `⌘9` | 快速貼上前 9 筆 |
| `⌃Tab` / `⌃⇧Tab` | 切換分頁 |
| `Esc` | 關閉視窗 |

### 首次使用

系統會要求授權「輔助使用」權限（用於模擬 Cmd+V 貼上），請到：

**系統設定 → 隱私權與安全性 → 輔助使用** → 開啟 Paster
